
To get the profile information on a user:
GET request to:
localhost:8080/profile/lelo
(lelo is a param)

Example Response for lelo =>
{
    "_id": "661f3241d1a87f3f7b938539",
    "displayName": "lelo",
    "username": "661f3241d1a87f3f7b938537",
    "videos": [
        <"video urls">
    ],
    "points": 0,
    "savedSkills": [],
    "__v": 0
}

To save a video:
POST request to
localhost:8080/savedSkills/lelo
body: { video url, title, subject }
(note: lelo is a param)
Reponse => { url, title, subject }


To update a user's points to watch a video:
A PATCH request to
/usePoints
body: user (note: I am expecting the whole object, but only need the id. Let me know what you end up using)
Response => {sufficianPoints : <boolean>} {profile} (note: the profile will have the un-updated points on it)
